# 如何免费使用大模型

## 方式一：使用 gpt4free（推荐）

### 1. 安装

```bash
# 进入项目目录
cd /workspace/gpt4free

# 安装依赖
pip install -r requirements.txt
```

### 2. 简单使用（Python）

```python
from g4f.client import Client

# 初始化客户端
client = Client()

# 发送请求
response = client.chat.completions.create(
    model="gpt-4o",  # 可以使用其他模型
    messages=[
        {"role": "system", "content": "你是一个乐于助人的助手。"},
        {"role": "user", "content": "介绍一下人工智能"}
    ],
)

# 打印回复
print(response.choices[0].message.content)
```

### 3. 启动 Web 界面

```bash
python -m g4f.gui
```

然后访问 http://localhost:8080

### 4. 可用的模型

项目支持多种模型，包括：
- GPT-4/5 系列
- Claude 系列
- Gemini 系列
- DeepSeek
- Qwen
- 等等
