# 后续实时样本学习流程

当前 redbook CLI 已安装，但本机没有可用小红书登录 Cookie，因此暂时不能抓实时笔记样本。

以后登录可用后，按这个流程更新知识库：

1. 搜索品类关键词，例如 `低帮鞋`、`训练鞋`、`手机壳推荐`、`通勤包`。
2. 分别看热门、最新、图文、视频。
3. 每个关键词只取少量高互动笔记，避免频繁访问。
4. 记录标题结构、封面大字、首图画面、正文结构、评论区购买疑问。
5. 不复制原文，不保存大段正文，只沉淀结构。
6. 按类目更新 `04_category_templates.md`。

样本抽取字段：

- keyword
- note_type
- title_pattern
- cover_hook
- first_image_role
- body_structure
- comment_objections
- conversion_trigger
- compliance_risk

沉淀原则：

- 学结构，不抄内容。
- 学用户疑问，不编体验。
- 学封面点击点，不制造虚假承诺。
